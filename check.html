Step 4: Create Database (If Needed)

Method 1: Using SQL File
  1. Open phpMyAdmin: Click here
  2. Click "SQL" tab at the top
  3. Open database_setup.sql file from your project folder
  4. Copy ALL content and paste in phpMyAdmin
  5. Click "Go" button

Method 2: Manual Creation
  1. Open phpMyAdmin
  2. Click "New" on left sidebar
  3. Database name: servicefinder
  4. Collation: utf8_general_ci
  5. Click "Create"
  6. Then run SQL from database_setup.sql
